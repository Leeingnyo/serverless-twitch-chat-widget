방송 도구에 추가하기

방송도구마다 달라서 안내가 어렵습니다.

단순 화면 공유를 하시는 중이라면 viewer.html 을 인터넷 브라우저로 여신 후 화면에 놓으세요.

OBS studio의 경우 브라우저 컴포넌트를 추가하시고 로컬 파일에 체크하신 뒤 viewer.html을 찾아서 넣으시면 됩니다.

XSplit도 비슷하게 viewer.html을 찾아서 넣으시면 됩니다 (로컬 파일이 있는지 모르겠으나 없다면 file:///~~ 로 넣으시면 됩니다).
